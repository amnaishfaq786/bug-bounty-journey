# Lab 04 - URLs and Parameters

## Objective

Understand how URLs are structured, determine which URL components are transmitted to the server, and practice modifying query parameters using Burp Suite Repeater.

---

## Environment

- Kali Linux
- Burp Suite Community Edition
- Chromium Browser
- Postman Echo (https://postman-echo.com)

---

## Labs Included

1. URL Components
2. Fragment Test
3. Query Parameters
4. Parameter Tampering

---

## Skills Learned

- Identifying URL components
- Understanding browser vs server processing
- Reading query strings
- Modifying HTTP requests
- Using Burp Repeater
- Verifying server responses

---

## Conclusion

This lab demonstrated that clients can modify HTTP requests before they reach the server. Query parameters are fully controlled by the client, while URL fragments remain inside the browser and are never transmitted.
